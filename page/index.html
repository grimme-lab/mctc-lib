This library supports reading and writing of the following formats:

  • xyz with extensions
  • Turbomole’s coord
  • connection table files: molfile, structure data format
  • Vasp’s POSCAR format
  • a subset of PDB format
  • DFTB+ general format
  • Gaussian external format
  • Chemical JSON format
  • QCSchema JSON format
  • Pymatgen JSON format
  • FHI-aims geometry.in
  • Q-Chem molecule format
