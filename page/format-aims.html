FHI-aims geometry.in format Chemical JSON Connection table format Gaussian external format DFTB+ general format Protein data bank (PDB) format Q-Chem molecule format QCSchema JSON Turbomole's coordinate data group Vasp's POSCAR format xyz format

Specification

Format used by FHI-aims program. Atoms are specified by atom or atom_frac keyword followed by three real numbers and an character identifier. Lattice parameters are given with the lattice_vector keyword followed by three real numbers.
