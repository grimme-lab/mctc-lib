Pymatgen JSON

  1. Formats
  2. Pymatgen JSON
  • Formats
FHI-aims geometry.in format Chemical JSON Connection table format Gaussian external format DFTB+ general format Protein data bank (PDB) format Pymatgen JSON Q-Chem molecule format QCSchema JSON Turbomole's coordinate data group Vasp's POSCAR format xyz format

Specification

Note

Reference

Pymatgen formatted JSON files are identified by the extension pmgjson and parsed following the Molecule or Structure format.
